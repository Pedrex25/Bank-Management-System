# Bank Account System in C

## Project Purpose

This project aims to help learners become more familiar with the C programming language by building a simple Bank Account System. It focuses on using core C concepts such as strings, arrays, structures, and functions to create an application that manages bank account data.

## Project Overview

- Create Account — Add new bank accounts with user details.
- Check Amount — View the current balance of an account.
- Transfer Money — Send money from one account to another.
- Login Functionality — Simulate user login to access their account.

## Learning Goals

- Understand and apply C structures to model complex data.
- Work with arrays and strings for storing user information.
- Implement functions to encapsulate different behaviors.
- Practice memory management and data manipulation.
- Build a simple command-line interface (CLI) for user interaction.
